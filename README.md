# chafan_website

使用 vue.js + element UI 编写的一个静态网站，后期会加入更多的元素进去。

## 项目安装

```
npm install
```

### 编译运行

```
npm run serve
```

### 全局安装vercel

```
npm i -g vercel  
```

### 部署到 vercel

```
vercel --prod
```

部署到vercel 需要到vercel 官网认证邮箱
